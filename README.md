**ClimaCast**

ClimaCast is a Streamlit app that lets you explore 40 years of historical weather data for any latitude and longitude. Using NASA’s POWER API, it retrieves temperature, wind, and precipitation data for a specific calendar day over the past four decades, converts metrics to Fahrenheit and MPH, classifies conditions (e.g., “Very Hot,” “Very Windy”), and visualizes trends with interactive Plotly plots. The app also calculates probabilities of extreme conditions and allows exporting the full dataset to CSV. Install dependencies with pip install streamlit requests plotly pandas and run the app with streamlit run test.py. This tool provides historical trends, not forecasts.
